### Overview 

  ChatUp is a persistent messaging application that runs on Node.js. It's designed to be secure and easy to use. This app allows multiple users to chat together in real-time using web sockets,
 
### Technologies used  

- Express
- Socket.io
- Handlebars
- Passport 
- Bcrypt
- Node.js
- MySql
